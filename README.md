# Power Comfort Institute (電力ゆとり研究所)

AI・EV時代の電力不安を解消するための、電力レジリエンス・シミュレーターです。

## 概要

本プロジェクトは、気象庁AMeDASのリアルタイムデータと、住居タイプ・ライフスタイル・所有装備（ポータブル電源、EV等）を組み合わせ、災害時や電力逼迫時の「電力ゆとり」を数値化します。

### 主な機能

- **独自算出ロジック**: 4つの主要変数（Housing, Dependency, Environment, Backup）から「電力ゆとり指数」を算出。
- **権威階層 (Authority Tiers)**: NASAや各公的機関のデータを参照し、情報の信頼性を階層化。
- **動的シナリオ**: 停電、冬季、台風など、特定の災害状況下でのリスクを見守ります。
